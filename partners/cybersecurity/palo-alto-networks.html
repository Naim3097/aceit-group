Next-Generation Firewall

Blocks traffic based on administrator-defined restrictions while protecting against modern threats like advanced malware, application-layer attacks, malicious URLs, and more.

Software that runs all Palo Alto Networks Next-Generation Firewalls. Leverages App-ID, Content-ID, Device-ID, and User-ID technologies with inline machine learning to control application use across all devices.
